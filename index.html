Cupcake Store

The Cupecake store was an exercise where we had just learned how to create links inside of our main html page. We then were instructed to create three different cupcakes and have each of them on a different page where the link was on the main page.

Resort

The Resort page was an exercise where we were to create a resort. The resort page was for a customer so we were supposed to give the customer a couple of options on the styling that they wanted done. For this we were taught how to branch. The branching made it so we could copy the master branch and be able to change the code around to try something else, all without changing the master branch.
